# Для чего было создано?
У драйверов видеокарт NVIDIA, есть очень много параметров (двордов), которые могут быть настроены реестр. Эти параметры зачастую скрытые и просто так их не увидеть, без стороннего софта. Но увидеть их имена, это лишь первая проблема, вторая же - правильно установить значение: <br> <br>
Есть дворды в имени которых есть Enable/Disable, так что значение для них, это 0x00000000 или 0x00000001. <br>
Если же в имени дворда нету Enable/Disable, то приходится работать с битфилдами, которые берутся с документации NVIDIA (со слива). <br>
- Битфилды - диапазон битов, к примеру 1:0, то есть бит 0 и бит 1 => 2 бита в общем. <br>
- Битмаска - готовое значение для дворда в двоичной системе исчесления, к примеру 0000 0000 0000 0000 0000 0000 0000 1111. <br>

Короче говоря, что бы рассчитать битмаску, нужно приложить время, nvtweak позволяет это все автоматизировать, путем парсинга документации, позволяя расчитывать значение для двордов и устанавливать его, очень легко и быстро.
Кроме этого, nvtweak предоставляет множество функций, для работы с документацией NVIDIA и двордов.
